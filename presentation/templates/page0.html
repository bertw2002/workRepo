INTRODUCTION



What exactly is CoffeeScript?

  • 'CoffeeScript is just Javascript'

  • Offers simpler syntax
    • Exposes the good parts of Javascript in a simple way
  • Uses elements of Python, Ruby, and Haskell
    • White spaces matter

    • Why CoffeeScript?

  • The compiled output will be readable and pretty-printed

  • Faster/as fast of Javascript
    • No delay when compiling into JS, no interpretation at runtime
  • Syntax used is easier to read and more efficient

    • How to install:

  • You need to have the latest version of Node.js In terminal

  • Input in terminal: npm install --global coffeescript
    • Makes commans 'coffee' and 'cake' usable. (compilers)
  • Then input: npm install --save-dev coffeescript
    • Allows CoffeeScript to be downloaded locally in your project
